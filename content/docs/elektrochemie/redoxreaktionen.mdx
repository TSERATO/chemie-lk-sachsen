## Definitionen

| Begriff | Definition |
|---|---|
| Oxidation | Abgabe von Elektronen (OIL) |
| Reduktion | Aufnahme von Elektronen (RIG) |
| Oxidationsmittel | Nimmt Elektronen auf, wird reduziert |
| Reduktionsmittel | Gibt Elektronen ab, wird oxidiert |

## Redoxgleichungen aufstellen

**Schema:**
1. Oxidations- und Reduktionshalbreaktion trennen
2. Elektronen ausgleichen
3. Ladungen ausgleichen (H⁺ oder OH⁻)
4. Wassermoleküle ergänzen

## Beispiel: Zink in Kupfersulfat-Lösung

$$
\text{Zn} + \text{Cu}^{2+} \rightarrow \text{Zn}^{2+} + \text{Cu}
$$
